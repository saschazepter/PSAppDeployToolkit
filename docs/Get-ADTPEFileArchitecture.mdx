## SYNOPSIS

Determine if a PE file is a 32-bit or a 64-bit file.

## SYNTAX

### Path

```powershell
Get-ADTPEFileArchitecture -Path <String[]> [-PassThru] [<CommonParameters>]
```

### LiteralPath

```powershell
Get-ADTPEFileArchitecture -LiteralPath <String[]> [-PassThru] [<CommonParameters>]
```

### InputObject

```powershell
Get-ADTPEFileArchitecture -InputObject <FileInfo> [-PassThru] [<CommonParameters>]
```

## DESCRIPTION

Determine if a PE file is a 32-bit or a 64-bit file by examining the file's image file header.

PE file extensions: .exe, .dll, .ocx, .drv, .sys, .scr, .efi, .cpl, .fon

## EXAMPLES

### EXAMPLE 1

```powershell
Get-ADTPEFileArchitecture -FilePath "$env:windir\notepad.exe"
```
