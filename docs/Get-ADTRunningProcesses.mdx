## SYNOPSIS

Gets the processes that are running from a list of process objects.

## SYNTAX

```powershell
Get-ADTRunningProcesses [-ProcessObjects] <ProcessDefinition[]> [<CommonParameters>]
```

## DESCRIPTION

Gets the processes that are running from a list of process objects.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-ADTRunningProcesses -ProcessObjects $processObjects
```

Returns a list of running processes.
If nothing is found nothing will be returned.

## PARAMETERS

### -ProcessObjects

One or more process objects to search for.
